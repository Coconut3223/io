    javascript¶

    • let 创建变量和对象

    变量 & 对象

    对象可以直接修改整个，也可以修改单个属性值，可以动态添加属性

    let var = "aaa"  // 创建变量
let obj = {  // 创建对象
    name: "aaa",  // 属性用冒号
    size: 123,
    bool: true,
}

    == 等于 & === 绝对等于

    • == 等于 只管 值

    • === 绝对等于 不仅管 值 ， 还管 类型

    >>> let num = 2;
>>> let str = '2';

>>> num == str
true
>>> num === str
false
